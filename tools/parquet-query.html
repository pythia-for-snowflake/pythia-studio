🦆 Parquet Query

Requête SQL locale sur n'importe quel fichier Parquet distant — DuckDB WASM, zéro backend, zéro crédit Snowflake

URL Parquet
Presets
SQL — utilisez $URL ou l'URL directement entre guillemets

Exemples : DESCRIBE SELECT * FROM read_parquet($URL) LIMIT 1  ·  SELECT COUNT(*) FROM read_parquet($URL)
