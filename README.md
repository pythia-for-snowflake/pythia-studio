# 🔮 pythia-studio

Browser-based tools for the [pythia-for-snowflake](https://github.com/gcwxfcchvz-source) suite.  
No build step — open any `.html` directly or serve via GitHub Pages.

**→ [gcwxfcchvz-source.github.io/pythia-studio](https://gcwxfcchvz-source.github.io/pythia-studio)**

---

## Tools

| Tool | File | Description |
|------|------|-------------|
| Palette & Title Generator | `palette-title-generator.html` | Signature color → harmony → semantic tokens → app title PNG export |
| Env Generator | `env-generator.html` | conda env · dependencies · pre-commit hooks · commit convention |
| Doc Hub — online | `hub-online.html` | Curated live docs: Snowflake, Snowpark, Streamlit, Python, Anthropic |
| Doc Hub — offline | `hub-offline.html` | Manage local wget mirrors of API docs |

---

## Stack

Static HTML · CSS · vanilla JS — no dependencies, no framework, no build.
